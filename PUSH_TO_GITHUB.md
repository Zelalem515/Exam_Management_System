# Push to GitHub

Your project is ready to push to GitHub. Follow these commands:

## Step 1: Initialize Git (if not already done)
```bash
git init
```

## Step 2: Add All Files
```bash
git add .
```

## Step 3: Create Initial Commit
```bash
git commit -m "Initial commit: PERN Stack Exam Management System"
```

## Step 4: Add Remote Repository
```bash
git remote add origin https://github.com/Zelalem515/Exam_Management_System.git
```

## Step 5: Push to GitHub
```bash
git push -u origin main
```

## Verify Push
Visit: https://github.com/Zelalem515/Exam_Management_System

---

**Note:** Make sure you have Git installed and are authenticated with GitHub before pushing.
